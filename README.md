# Astro SaaS Landing — Двуязычный лендинг EN/RU

Современный, быстрый, SEO-оптимизированный SaaS/agency лендинг на **Astro 4**. Светлый минималистичный дизайн с поддержкой двух языков — английского и русского.

---

## Стек и технологии

- **[Astro 4](https://astro.build/)** — статическая генерация, нулевой JavaScript по умолчанию
- **TypeScript** — строгая типизация по всему проекту
- **CSS Custom Properties** — дизайн-токены для единой системы стилей
- **Inter** — современный шрифт для профессиональной типографики
- **JSON локали** — отдельные файлы переводов для каждой страницы и языка

---

## Возможности проекта

### Мультиязычность
- Полная поддержка **EN** и **RU** с раздельными маршрутами (`/en/...` и `/ru/...`)
- Система локализации через `src/i18n/utils.ts` с `buildPath()` для генерации корректных URL
- Переключатель языков в навигации
- Раздельные JSON-файлы для каждой страницы и языка в `src/locales/pages/`

### Страницы
| Маршрут | Описание |
|---------|----------|
| `/` | Редирект на `/en/` |
| `/en/` / `/ru/` | Главная страница |
| `/en/about` / `/ru/about` | О нас |
| `/en/features` / `/ru/features` | Возможности продукта |
| `/en/products` / `/ru/products` | Продукты |
| `/en/pricing` / `/ru/pricing` | Тарифы |
| `/en/contact` / `/ru/contact` | Контакты |
| `/404` | Страница ошибки |

### Компоненты
- **HeroLicense** — премиум-герой с фоновым изображением и статистикой
- **HeroSection** — универсальный герой для внутренних страниц
- **ProductOverview** — сравнительные карточки с зелёной/красной цветовой схемой
- **ProductStrategies** — карточки стратегий с тегами и блоком советов
- **ProductWhy** — сетка преимуществ с CTA
- **HighlightCards** — карточки возможностей с цветовыми акцентами
- **SectionIntro** — вводная секция с карточкой и CTA
- **TermsList** — аккордеон условий и терминов
- **TrustPlatform** — карточки с данными о платформе
- **TrustRoutine** — блок доверия с шагами
- **ContactPremium** — форма контактов с teaser-блоком и showcase
- **FAQ** — аккордеон часто задаваемых вопросов
- **ContentArticle** — статейный блок для страницы "О нас"
- **Header** — фиксированная навигация с переключателем языков
- **Footer** — тёмный футер с колонками и callout-блоком

### SEO
- JSON-LD структурированные данные (`SeoSchema`) для каждой страницы
- Open Graph теги (`SeoOG`, `SeoOGHome`) для соцсетей
- `canon.json` — централизованное управление canonical/hreflang URL
- `sitemap.xml` + `robots.txt`
- Web App Manifest для PWA

### Дизайн-система
- Светлая минималистичная палитра: белый/`#F8FAFC` фон, тёмный текст `#0F172A`, акцентный `#6366F1`
- CSS Custom Properties как дизайн-токены: `--color-primary`, `--color-text`, `--color-border`, `--shadow-sm`, `--shadow-indigo` и др.
- Адаптивная верстка для мобильных, планшетов и десктопов
- Плавные переходы и hover-эффекты

## Структура проекта

```
src/
├── components/       # Astro-компоненты
│   └── seo/          # SEO-компоненты
├── i18n/
│   └── utils.ts      # Утилиты локализации
├── layouts/          # BaseLayout, PageLayout
├── locales/
│   ├── common/       # Общие переводы (footer и др.)
│   └── pages/        # Переводы по страницам (en.json / ru.json)
├── pages/
│   ├── en/           # Английские страницы
│   └── ru/           # Русские страницы
├── styles/
│   └── global.css    # Глобальные стили и токены
└── utils/            # env, canon, i18n утилиты
```

```env
PUBLIC_REF=https://yourdomain.com
PUBLIC_BRAND=YourBrand
```
